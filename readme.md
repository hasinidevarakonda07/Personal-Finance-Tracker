# Personal Finance Tracker

## Overview

Personal Finance Tracker is a web-based application designed to help users efficiently manage their personal finances. The system enables users to track income and expenses, monitor spending habits, set budgets, and generate financial insights through interactive reports and visualizations.

The goal of this project is to promote better financial planning and provide users with a clear understanding of their financial activities.

---

## Features

### Expense Management

* Add, edit, and delete expenses
* Categorize expenses by type
* Track daily, weekly, and monthly spending

### Income Tracking

* Record multiple income sources
* Monitor cash flow
* Calculate net savings

### Budget Planning

* Set monthly spending limits
* Compare actual expenses with budget goals
* Receive overspending alerts

### Financial Analytics

* Category-wise spending analysis
* Monthly financial summaries
* Savings and expenditure trends
* Interactive charts and reports

### User-Friendly Dashboard

* Overview of financial status
* Recent transactions display
* Budget progress indicators

## System Architecture

1. Users record income and expenses.
2. Financial data is stored in the database.
3. The system processes and categorizes transactions.
4. Analytics modules generate spending insights.
5. Results are displayed through interactive dashboards and charts.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/personal-finance-tracker.git
```

Navigate to the project directory:

```bash
cd personal-finance-tracker
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
